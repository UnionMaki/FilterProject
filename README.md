# 🎬 Video Enhancer — улучшение качества видео в реальном времени

Программа на C++ для повышения качества видеопотока в реальном времени.  
Подходит для обработки как локальных файлов, так и сетевых потоков.

---

## ✨ Возможности

- 📁 **Локальный режим** — тестирование фильтров на готовых видеофайлах
- 🧪 **Тестовый режим** — полная обработка видео с последующим воспроизведением *(для проверки качества фильтров)*

---

## 🎮 Режимы работы

### 1. Тестовый режим (оффлайн-обработка)

Видео сначала полностью обрабатывается, затем проигрывается.  
**Используется для:** проверки и настройки фильтров.

```bash
my_app.exe -f C:\videos\input.avi
```

### 2. Режим реального времени (поток)

Обработка кадров происходит мгновенно, без накопления.

```bash
my_app.exe -f rtsp://192.168.1.100/stream
```

---

## 🧪 Эмуляция видеопотока с помощью FFmpeg

Если нет реального потока, можно создать тестовый с помощью FFmpeg:

```bash
ffmpeg -re -stream_loop -1 -i input.avi -c copy -f rtsp rtsp://localhost:8554/mystream
```

Затем подключитесь к нему:

```bash
my_app.exe -f rtsp://localhost:8554/mystream
```
